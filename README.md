#西瓜輸入法

西瓜輸入法為一個特別為了觸控介面所設計的中文輸入法
更少的按鍵需求，讓使用者有效的提升打字效率

##特殊功能 Feature

* 滑動連續刪除<br>
* 聲母、韻母鍵盤配置<br>
* 中文數字鍵盤<br>
* 顏文字<br>
* 滑動調整英文大小寫<br>
* 無縫切換「注音輸入法」與「漢語拼音輸入法」<br>
